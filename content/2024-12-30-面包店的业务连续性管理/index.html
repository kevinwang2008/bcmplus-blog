开场：面包店也需要业务连续性管理 [1]

有人说，大企业就是放大版的面包店。买面粉、鸡蛋和糖叫作采购，和面、塑形和烤制叫作生产，运到面包店叫物流，把它们卖出去叫作销售。这是一个不精确但挺有意思的说法，大致是想说，面包店“麻雀虽小，五脏俱全”，具备了现代企业的主要要素。

那么，我们为什么不从一家面包店开始理解怎么做业务连续性管理呢？事实上，这完全是可行的。

一方面，并非只有大企业才需要业务连续性管理，中小微企业同样需要。刚过去不久的疫情已突显了这一点，不管是多达数万员工的大型企业，还是只有3、5名员工的面包店，都无可避免会受到停业的威胁。

另一方面，从一家面包店开始考虑业务连续性管理，可以让我们从大企业那些繁杂的管理活动中走出来，直接抓住业务连续性管理中最本真的部分。

那么，假如你是一家面包店的老板，你的资源和人员比较有限，很容易受到各种（即使是轻微的）糟糕事件的影响。例如，受到意外事件（如自然灾害、周边环境事故等）影响临时停业造成财务损失，严重的话可能很难再恢复正常营业，甚至永久关店。显然， 你的面包店需要进行业务连续性管理 。

那么，一家面包店该怎么进行业务连续性管理呢？

我们可以将面包店的业务连续性管理工作分为3步： 第一步，风险和恢复能力评估； 第二步，开发并落实应对方案； 第三步，监测变化并及时更新。

当然，首先，我们需要说说这家面包店的基本情况 [2] ： 店铺面积 ：约50平方米（月租金20000元）； 营业时间 ：每天12小时（8：00 – 20：00）； 客户群体 ： 上班族（偏好方便携带的早餐、下午茶）； 家庭主妇/主夫（偏好健康、新鲜的家庭分享面包）； 健康人群（寻找全麦、低糖、无添加等特色产品）； 差异化定位： 与连锁品牌相比，提供更定制化、更高品质的产品； 与超市相比，突出新鲜感、健康配方和手工制作； 覆盖范围 ：所在小区及周边1-3公里； 日均顾客数量 ：125人； 人均消费 ：27元/人； 产品毛利率 ：60%； 经营成本 ：月水电杂费为3000元，月设备等折旧费用为3000元； 员工方面： 共6名，包括店长1名（ 当然，在这个案例中，店长可由你这位老板兼任 ），2名烘焙师，3名（其中1名为兼职）收银/服务员，月总工资为24000元。

==== 第一步，风险和恢复能力评估 ====

作为面包店的老板，可以从回答以下3个问题开始业务连续性管理： 我的生意面临哪些可能导致停业的风险？ 我的生意到底可以承受多长时间的停业？ 一旦中断，业务需要多长时间恢复正常？

我们依次来看这3个问题：

第1个问题， 面包店面临哪些可能导致停业的风险？

经过认真思考，我们会得出，导致面包店停业的主要风险包括： 供应链扰断: 面包店依靠供应商提供包括面粉、糖、乳制品等在内的各种食材。供应链扰断（扰乱或中断）会影响面包店的正常运营，无论这些扰断是由运输、供应商经营问题还是 全球食品供应链扰断造成的 ； 设备故障: 面包店的关键设备一旦不正常工作，例如烤箱故障或制冷故障，面包店就可能无法进行生产或储存产品； 人员短缺: 面包店，尤其是小型面包店，可能严重依赖一些关键员工。由于疾病、个人/家庭紧急情况或其他原因导致的意外缺勤可能会扰乱正常运营； 自然灾害: 在自然灾害（例如城市内涝、台风和地震等）风险较高的地区，面包店和其它企业都面临着特定的挑战，这些自然灾难可能造成断电、建筑物损坏或员工和客户无法进入面包店； 法律法规变化: 食品/卫生/劳动/环保法律法规或当地监管环境的变化可能会影响运营，面包店需要做好准备以适应这些变化。

第2个问题， 面包店到底可以承受多长时间的停业？

我们从财务方面进行一些简单测算（ 以下测算基于面包店的基本情况 ）：

面包店的 月固定成本 = 月租金 + 员工工资 + 水电杂费 + 折旧费 = 20000 + 3000 + 3000 + 24000 = 50000元；

面包店 每天的销售额 = 人均消费 × 日均顾客数量 ＝ 27元/人 × 125人 = 3375元， 面包店的 月销售额 = 每天销售额 × 每月天数 = 3375元/天 × 30天 = 101250元， 面包店的 月毛利润 = 月销售额 × 毛利率 = 101250元 × 60% = 60750元；

根据以上计算，面包店的 月利润 = 月毛利润 – 月固定成本 = 60750 – 50000 = 10750元， 相应的，面包店的 年利润 为：10750 × 12 = 129000元。

此外，面包店的 月盈亏平衡点 = 月固定成本 ÷ 毛利率 = 50000元 ÷ 60% ≈ 83333元； 也就是说， 面包店每月至少需要正常运营 83333元 ÷（3375元/天） = 24.7天 ≈ 25天才可以保障不亏损 ； 所以， 面包店最多可以承受 （30 – 25） = 5天的停业时间 。（即每月停业超过5天，当月就会从赢利转为亏损状态）

当然，根据风险承受能力，可以将面包店可以停业的目标底线设置的更短或更长一些，比如说：3天或者8天。

也就是说，可以设定面包店的 恢复时间目标（Recovery Time Object，RTO） 为3天或者8天。

第3个问题， 一旦中断，业务需要多长时间恢复正常？

这个问题的回答会稍复杂一些，因为需要将营业中断情景细分并具体化，并基于当前能力进行评估。

比如，对于供应链扰断，需要逐一列出每个供应商，分别评估其一旦断供，面包店需要多长时间才能将业务恢复正常（如果没有备用供应商，可能需要等待该供应商恢复正常，或者临时寻找合格的供应商进行采购）。对于设备故障等中断风险，也需要逐一分析评估。

应该找出那些超出设定的面包店恢复时间目标的情况，这是接下来业务连续性管理工作的重点。

回答好以上3个问题是所有业务连续性管理工作的基础，它们可以帮助了解面包店所面临的具体营业中断风险、设定业务运营（可容忍停业）的底线时间目标，以及面包店当前的恢复能力。

==== 第二步，开发并落实应对方案 ====

在开发并落实应对方案时，同样可以通过回答以下3个问题来理清思路： 我将采用哪些策略来应对已识别的风险？ 我准备怎么落实所选择的风险处置策略？ 我怎么验证或者确认恢复方案是有效的？

我们仍然依次来看这3个问题：

第4个问题， 我将采用哪些策略来应对已识别的风险？

对于前述已识别的风险，我们可以逐项看看该怎么应对。

对于 供应链扰断， 可以采用的策略有 ： 多元化供应商 : 考虑从多个供应商处采购食材，降低单一供应商遇到问题时造成停业的风险； 优化库存储备 : 保留一定数量的缓冲库存，有助于应对供应链短时间的中断； 本地采购 : 尽可能进行本地采购，减少与长途运输和全球供应链中断相关的风险。

对于 设备故障 ，可以采用的策略有： 定期维护 : 对所有设备进行定期检查和维护，这种主动性维护可以减少设备意外故障的发生； 备用设备 : 考虑对关键设备进行备份。例如，备用烤箱或备用电源在故障期间的价值怎么高估都不算错； 培训 : 培训员工处理轻微设备故障，这样他们可以解决小问题或知道何时需要专业帮助。

对于人 员短缺 ，可以采用的策略有： 交叉培训 : 培训员工承担多个岗位，这样，如果关键员工缺勤，另一位员工可以补位。在面包店，烘焙师、收银/服务员和店长的岗位都应有替补角色； 灵活的工作计划 : 提供灵活的工作时间表或兼职岗位，以吸引更多员工； 临时工派遣 : 与人员招聘/临时用工或者其它面包店建立关系，在人员意外短缺时由他们帮助提供临时工。

对于 自然灾害 ，可以采用的策略有： 保险 : 可以考虑对本地区普遍存在的自然灾害购买足够的保险； 防灾计划 : 制定防灾计划，概述自然灾害之前、期间和之后应采取的行动，应包括疏散程序，沟通策略和恢复行动； 备用电源 : 投资临时电源在停电期间提供电力；

对于 法律法规变化 ，可以采用的策略有： 随时了解情况 : 加入当地商业协会、商会或与所在社区、街道保持联系，随时了解可能的法律法规变化； 定期培训 : 定期为员工举办培训课程，以确保每个人都了解并遵守现行法律法规； 咨询 : 考虑在法规复杂或经常变化的领域（例如卫生法规或劳动法）聘用或咨询专家。

第5个问题， 我准备怎么落实所选择的风险处置策略？

一旦选定了风险处置策略，接下来就是将这些策略付诸实践。实施是将选定的策略转化为保护面包店尽可能少受扰断负面影响的现实世界实践，具体可以包括：

准备并分配资源 : 确保拥有实施选定策略所需的资金、人员和设备，这意味着需要为备用设备留出预算，或雇用更多人员进行交叉培训等； 制定行动计划 : 对于每种策略，编写详细的行动计划，包括需要做的事情、由谁负责、所需的资源以及完成的时间表； 培训和练习 : 为员工提供有关新程序或设备的培训。例如，如果决定对员工进行交叉培训，请安排培训课程，以便他们可以学习新岗位应掌握的技能； 沟通 : 让所有人都了解情况。告知员工、供应商，甚至客户（如相关），我正在为确保业务连续性进行工作调整； 文档化 : 记录所有工作调整、新流程和培训课程，这些文档可以合并为一个业务连续性计划文件。这不仅可以作为工作记录和必要的备案，还可以确保未来的新员工能快速成长或相关方快速了解情况。

第6个问题， 我怎么验证或者确认恢复方案是有效的？

一旦一切就绪，就需要通过演习或模拟来测试恢复方案的有效性。例如，模拟供应链中断，来看看多元化供应商或优化库存储备是否能够应对这种情况。

回答好这3个问题是业务连续性管理工作的核心，它们可以帮助确定，一旦面临营业中断，是否已准备好必要的资源（财务、人力/技能、技术）并支持面包店的业务恢复正常。

==== 第三步，监测变化并及时更新 ====

业务连续性管理不是一次性的工作。随着面包店的发展以及外部因素的变化，面包店的业务连续性管理也需要做出相应的调整。

仍然可以通过回答以下2个问题理清思路： 有什么内外部变化会影响面包店的运营？ 当前能力是否能够应对这些内外部变化？ 如果不能，面包店需要做哪些必要调整？

这两个问题的关键在于面包店所面临的停业风险是否有变化，以及面包店的业务连续性能力是否能够应对这些新变化，需要做的工作包括： 定期审查 : 对面包店当前的业务连续性能力定期审查以找到与面包店所面临风险不匹配的情况，这有助于发现可改进的方面； 收集反馈 : 在演练或事件（或未遂事件）后，收集员工、供应商和其他相关方的反馈，他们的意见对于完善业务连续性管理可能是无价的； 跟踪外部变化 : 留意可能影响业务连续性管理的外部因素，例如法律法规或监管环境的变化、新兴风险或市场上出现的新解决方案； 调整和更新 : 根据定期审查结论、所收集反馈以及外部变化情况，对面包店的业务连续性管理进行必要的调整。这可能意味着增加新方案、去掉过时的方案或调整现有策略以提高有效性。 再培训和练习 : 在对业务连续性管理进行重大调整后，要确保所有相关人员重新参加培训和练习； 文件变更 : 确保对业务连续性管理的所有变更，都能反映在必要的文档里。

通过不断审查和完善面包店的业务连续性管理，可以确保面包店已准备就绪，以应对各种挑战。

==== 小型企业业务连续性管理的特点 ====

以上，我以一家面包店为例，介绍了小型企业进行业务连续性管理的主要工作（包括3步和回答8个问题），不同规模和行业的企业都可以从这些工作中受益。

下面，我简要讨论一下小型企业业务连续性管理的特点：

  1. 小型企业业务连续性管理的挑战

由于预算和资源有限，小型企业更容易受到业务中断的影响。需要精心规划和实施业务连续性管理，将资源分配给关键功能，并避免不必要的支出。

小型企业的另一个挑战是快速的变化和对适应性的需求。小型企业需要更加敏捷，并具备快速发展以应对业务环境中的新挑战或变化的能力。主动性策略（主动进行业务连续性管理而不是在中断发生时被动应对）可增强不确定环境中的业务韧性。

此外，小型企业的团队规模较小，每个团队成员都非常重要。甚至一名员工的缺勤或失能也会扰乱业务流程。因此，有效的岗位替补策略和协调的应对计划变得至关重要。

  1. 小型企业业务连续性管理的收益

业务可持续性 : 健全的业务连续性管理可帮助小型企业应对危机，保持消费者信任和品牌声誉。 投资者和客户信任 : 运营韧性和主动规划可以增强投资者和客户的信心。 竞争优势 : 健全的业务连续性管理可以在危机期间使小型企业领先于竞争对手。

  1. 小型企业连续性管理的常见陷阱

未及时更新 : 对业务连续性管理及时更新可确保其实用性。 忽略较小风险 : 小型企业应考虑所有中断情景，无论其规模如何。 团队培训不足 : 经过适当培训的员工可以有效应对危机。

通过主动的业务连续性管理，各种规模和行业的企业都可以自信地应对中断情景。尽管资源有限，但业务连续性计划提供的明确路线图明确了要采取的步骤和资源分配。除了保障能在中断后生存下来，业务连续性管理还可以提供竞争优势并增强员工的能力，从而促进长期的业务增长。
